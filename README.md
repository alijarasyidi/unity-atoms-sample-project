# unity-atoms-sample-project

## Prerequisite
- Project using Unity version 2020.3.3f1
- About Unity Atoms (https://github.com/unity-atoms/unity-atoms)


## Description
This is sample projects to try using Unity Atoms system feature. As for content, it is a simple scoring system with requirements:
- For mobile phone
- Looks good on various mobile phone resolution
- Separate class for increasing & decreasing score
- Clicking *Increase Score* button increase score by *Score Increase Factor*
- Clicking *Decrease Score* button decrease score by *Score Decrease Factor*
- *Score* can't be bellow 0
- *Decrease Score* button can't be clicked when *Score* is 0
